# Day 2 - Prompt Engineering

## Objective
Learn the difference between a Lazy Prompt and an Engineered Prompt.

## Lazy Prompt

Explain Prompt Engineering.

## Engineered Prompt

You are an AI educator teaching complete beginners.
Explain Prompt Engineering in simple language.
Include:
* What Prompt Engineering is
* Why it matters when using AI tools like Claude
* One example of a weak prompt
* One example of an improved prompt
* Three practical benefits of writing better prompts

## Comparison

### Lazy Prompt Output
- Short and generic response
- Less detailed explanation
- Limited examples

### Engineered Prompt Output
- Detailed explanation
- Better structure
- Clear examples
- More useful output

## Key Learnings
- Better prompts produce better AI responses.
- Structured prompts improve output quality.
- Clear instructions lead to more useful results.

## Conclusion
Engineered prompts generate more detailed and valuable outputs than lazy prompts.
