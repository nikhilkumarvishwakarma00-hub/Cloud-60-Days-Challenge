# Day 8 - Environmental Health Analyzer Dashboard

## Objective
Build an Environmental Health Analyzer Dashboard using Claude AI.

## What I Built
A complete HTML dashboard that analyzes environmental health data, air quality, water quality, health impacts, and city comparisons.

## Features
- Interactive Dashboard
- AQI Analysis
- Water Quality Analysis
- Health Impact Assessment
- City Comparison
- Charts and Visualizations
- Responsive UI

## Key Learnings
- AI can generate complete web applications.
- Detailed prompts produce better outputs.
- Dashboard creation becomes much faster with AI assistance.
- HTML, CSS, JavaScript, and Chart.js can be generated together.

- Day 8.html

## Conclusion
This exercise demonstrated how AI can generate a functional and interactive dashboard application from a single prompt.
