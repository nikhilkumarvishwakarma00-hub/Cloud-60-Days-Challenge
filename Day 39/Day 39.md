# Day 39 - PDF Splitter & Merger

## Objective

Build an AI-powered PDF Splitter & Merger application that enables users to split, merge, preview, and download PDF documents through a browser-based interface.

---

## Features

- PDF Splitter
- PDF Merger
- PDF Preview
- Download Processed PDFs
- Drag-and-Drop File Upload
- Responsive Interface
- Local File Processing
