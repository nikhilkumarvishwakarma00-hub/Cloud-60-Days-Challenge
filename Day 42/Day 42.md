# Day 42 - Personal Financial Command Center

## Objective

Build an AI-powered Personal Financial Command Center that helps users monitor financial health, analyze spending, and simulate financial decisions.

---

## Features

- Financial Health Dashboard
- Income Tracking
- Expense Tracking
- Budget Management
- Savings & Investment Overview
- Financial Health Score
- AI Recommendations
- What-If Simulation
- Interactive Charts

---

## Workflow

1. Answered Claude's onboarding questions.
2. Generated the complete HTML application.
3. Opened the application locally.
4. Explored every financial module.
5. Reviewed the Financial Health Score.
6. Analyzed AI-generated recommendations.
7. Tested different What-If simulation scenarios.
8. Captured screenshots.
